3. **Financial Post**

4. **AP Associated Press**

## Notes

- Tous les logos doivent être en format PNG avec fond transparent (ou blanc selon le design)
- Dimensions recommandées: 200-300px de largeur
- Utiliser les logos officiels uniquement (pas de versions modifiées)
- Vérifier les droits d'utilisation pour chaque logo
- Les logos doivent être lisibles sur fond sombre (#1a1a2e)
